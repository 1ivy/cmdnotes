### Git:

#### what Github says:
```shell
#### - new repo from cmd line:
git init
git add README.md
git commit -m "commit message"
git branch -M main
git remote add origin <origin>
git push -u origin main

#### - push an exiting repo from cmd line:
git remote add origin <origin>
git branch -M main
git push -u origin main
```
